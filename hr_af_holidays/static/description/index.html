AF Public Holidays

This module adds a public holidays and shortened work days that applies to the employees of Arbetsförmedlingen. Please visit VIS website for more information.

Configuration

To configure this module, you need to:

  1. Go to Tekniskt > Resurs > Resursavgånger to see the list
  2. Observe dates and times.
  3. To create a new holiday press Skapa and follow instructions
  4. To delete a holiday check the row and click on Åtgärd > Ta bort.

Usage

  1. Go to Tekniskt > Resurs > Resursavgånger to see the list
  2. Observe dates and times.

Alternatively:

  1. Tekniskt > Resurs > Working Times > Global Leaves menu
  2. Observe dates and times.
